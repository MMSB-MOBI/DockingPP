Loader¶

DockingPP.loader module To load raw results into DockingHandler objects

loadZdock(zdock_results: str, nb_pose: int = - 1) → DockingPP.dockingHandler.DockingHandler¶

Load zdock results

Parameters

  • zdock_results (str) – path to zdock results file

  • nb_pose (int, optional) – number of poses to load. If -1, all poses will be loaded. Defaults to -1.

Raises

  • error.ZdockFormatError – Raise if error is detected in zdock format

  • error.IncompatiblePoseNumber – Raise if you try to load more poses than possible.

Returns

DockingHandler object for this zdock results.

Return type

DockingPP.dockingHandler.DockingHandler
